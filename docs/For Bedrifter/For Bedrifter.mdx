# For bedrifter

Linjeforeningen Enigma legger stor vekt på å etablere og opprettholde nære forbindelser mellom bedrifter og studentene på ITØK. Det finnes flere muligheter for bedrifter å engasjere seg med våre studenter. Vi er alltid ute etter å skape det beste opplegget, både for dere som bedrift og oss som studenter. Nedenfor presenteres en oversikt over noen av de mest vanlige mulighetene for å komme i kontakt med våre studenter. Ta gjerne  kontakt dersom noe av dette er av interesse, eller om dere ønsker å etablere kontakt med studentene våre på en annen måte.
