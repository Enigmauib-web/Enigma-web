# Om Enigma

Enigma er linjeforeningen for alle studentene ved sivilingeniørprogrammet Informasjonsteknologi og økonomi (ITØK) ved Universitetet i Bergen. 
 
Enigma ble grunnlagt høsten 2021 av det første kullet med ITØK-studenter. Formålet med linjeforeningen er å bidra til et godt læringsmiljø blant studentene, legge til rette for sosiale aktiviteter og fremme kontakten med næringslivet. I tillegg skal foreningen være med å støtte opp om studentenes interesser og krav ovenfor instituttet samt andre relevante organ. Alle studentene på ITØK regnes som medlemmer av Enigma, og foreningen ledes av styret bestående av åtte medlemmer.
